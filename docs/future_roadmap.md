# 🚀 My Chrome Guide: 중장기 거버넌스 및 플랫폼 로드맵

본 문서는 "My Chrome Guide"가 단순한 팁 제공 도구를 넘어, **글로벌 매크로 공유 플랫폼**으로 진화하기 위한 단계별 전략을 정의합니다.

---

## 📅 시계열 마일스톤 (Timeline Milestones)

### 🟢 1단계: 개인화 및 로컬 최적화 (완료)
*   **핵심**: 전적 팁 큐레이션 및 로컬 매크로 엔진 구축.
*   **주요 성과**:
    - 139+ 전문가 선별 팁 DB 구축.
    - JSON 기반 데이터 입출력(Import/Export) 구현 -> **데이터 이동성 확보**.
    - 100% XSS 방어 설계 및 온보딩 UX 완성.

### 🟡 2단계: 클라우드 동기화 및 '매크로 스토어' (Next)
*   **핵심**: 로컬을 넘어 서버 기반의 데이터 공유 체계 마련.
*   **주요 기능**:
    - **Cloud Sync**: Google 로그인 연동으로 기기 간 매크로 자동 동기화.
    - **Macro Explorer**: 우리가 직접 큐레이션한 '공식 매크로 팩'을 앱 내에서 원클릭으로 다운로드하는 스토어 인터페이스.
    - **K-Portal Expansion**: **네이버(Naver)** 메일, 카페, 쇼핑 및 카카오 등 국내 특화 서비스 전용 매크로 및 팁 패키지 공식 지원.
    - **Scenario Packs**: 특정 테마별 매크로 세트(예: "신입사원 패키지", "주식 투자자 패키지") 제공.

### 🟠 3단계: 커뮤니티 및 오픈 마켓 플레이스
*   **핵심**: 사용자가 공급자(Creator)가 되는 생태계 조성.
*   **주요 기능**:
    - **User Upload**: 개인이 만든 매크로를 플랫폼에 등록하고 별점/댓글로 피드백 수집.
    - **Leaderboard**: 주간 베스트 매크로 제작자 랭킹 시스템.
    - **Premium Macros**: 고도화된 자동화 스크립트를 유료로 거래할 수 있는 마켓플레이스(수익 모델).

### 🔴 4단계: AI 에이전트 결합 (Vision)
*   **핵심**: LLM(AI)이 매크로를 자동으로 생성하고 실행하는 지능형 비서.
*   **주요 기능**:
    - **Natural Language Macro**: "내일 오전 9시에 네이버 메일로 업무 일지 보내줘"라고 말하면 AI가 매크로 스텝을 자동 생성.
    - **Auto-Repairing**: 사이트 UI가 변경되어 매크로가 깨졌을 때 AI가 자동으로 감지하고 수정 제안.
    - **Visual Guide Mode**: 보안 프로그램이 감지되는 민감한 사이트(공공/은행 등)에서 자동 클릭 대신 **스포트라이트와 텍스트 가이드**만 제공하는 '안전 모드' 구현.

## 🛠️ 기술적 도전 과제 (Technical Challenges)

1.  **샌드박스 보안 (Security Sandbox)**: 타인이 만든 매크로를 실행할 때 개인 정보를 탈취하지 못하도록 엄격한 실행 권한 제어.
2.  **안정적인 백엔드 (Scalable Backend)**: 수만 명의 사용자가 동시에 매크로를 조회하고 다운로드할 수 있는 Firebase/Supabase 기반의 인프라.
3.  **UI 엔진 확장**: 팝업(380px)의 한계를 넘어선 넓은 대시보드 화면(`index.html` 확장) 개발.
4.  **Whale/Edge 호환성**: 크롬 기반의 타 브라우저(네이버 웨일 등) 전용 기능 API 연동 연구.

---

## 📈 비즈니스 목표 (Business Goals)

*   **리텐션 극대화**: 사용자가 직접 만든 데이터가 쌓일수록 타 서비스로의 이탈이 불가능한 **락인(Lock-in)** 효과 발생.
*   **지역화 시장 선점**: 글로벌 툴들이 놓치기 쉬운 **한국형 포털/쇼핑 환경**에 최적화된 독보적인 매크로 생태계 구축.
*   **확장성**: 브라우저 팁이라는 좁은 시장에서 '업무 자동화 솔루션'이라는 거대 B2B/B2C 시장으로의 피벗(Pivot).
